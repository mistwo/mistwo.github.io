其它

  • 无法访问 GitHub，可以尝试修改 hosts 文件，在这个网址获取新的 ip：点击这里
  • 可能用到的其它命令：

  • 配置 _config.yml 时注意冒号 : 之后需要空格

  • _config.yml 文件有好几处，注意区分

  • 开启目录：toc: true
  • 开启截断：在阶段处插入 <!-- more -->
  • 添加空行：<br>
  • 标签：tag: -xx -xx -xx
  • 删除文章：保证 _posts 文件夹非空，删除本地文件后，运行命令hexo clean
