四、注意事项

1. 关于配置

  • 可能需要额外配置 Path
  • 无法访问 GitHub Pages，可以尝试将 http: 修改为 https:，另外也有可能是校园网的原因
  • 无法访问 GitHub，可以尝试修改 hosts 文件，在这个网址获取新的 ip：点击这里
  • 在访问localhost:4000时遇到问题： WARN No layout: index.html

  • 配置 _config.yml 时注意冒号 : 之后需要空格

2. 主题细节

  • _config.yml 文件有好几处，注意区分

  • 开启目录：toc: true

  • 开启截断：在阶段处插入 <!-- more -->

  • 添加空行：<br>

  • 标签：tag: -xx -xx -xx

  • 删除文章：保证 _posts 文件夹非空，删除本地文件后，运行命令hexo clean
